2019 – 2020

Workstation-based System Interface Redesign

The Weather Company Max is a workstation-based system that is used by 90%+ of all local TV stations in North America to produce visualizations and graphics for broadcast.

I was responsible for providing a new look and feel for the user interface redesign:

  • overall color theme
  • all visual components
  • patterns
  • layouts
  • iconography

Updated the icon library, redesigning over 200 icons. This long overdue visual overhaul produced a stunning modern dark themed interface that received extremely positive feedback from end users (meteorologists).
